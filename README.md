# 🪙 CoinKeeper Bot — Асинхронный Telegram-трекер расходов

Легковесный Telegram-бот для быстрого учета личных финансов. Позволяет моментально фиксировать траты естественным текстом и просматривать статистику без необходимости навигации по сложным меню.

## 🚀 Попробовать вживую
Бот задеплоен на удаленном сервере и работает 24/7: **[@NTCoinKeeperBot](https://t.me/NTCoinKeeperBot)**

## ⚡️ Основной функционал
* **Быстрая запись трат:** Отправьте сообщение в формате `150 кофе` или `1500 яндекс такси`, и бот автоматически распарсит текст, сохранив сумму и категорию.
* **Валидация данных:** Встроенная защита от некорректного ввода (отсутствие категории, ввод букв вместо чисел).
* **`/getstats`** — Вывод общей суммы всех записанных расходов пользователя.
* **`/getrecent`** — Просмотр детализированной истории последних 5 транзакций.

## 🛠 Технологии и стек
* **Язык:** Python 3.x
* **Фреймворк:** `aiogram 3.x` (асинхронная архитектура)
* **База данных:** SQLite3 (использование параметризованных запросов для защиты от SQL-инъекций)
* **Инфраструктура:** Linux, `systemd` (работа в виде демона), `RotatingFileHandler` (ротация логов)

## 💻 Локальный запуск (для разработчиков)

1. Склонируйте репозиторий:
`git clone https://github.com/NonTock/CoinKeeperBot.git`
`cd CoinKeeperBot`

2. Создайте и активируйте виртуальное окружение:
`python -m venv venv`
# Для Windows:
`venv\Scripts\activate`
# Для Linux/macOS:
`source venv/bin/activate`

3. Установите зависимости:
`pip install -r requirements.txt`

4. Создайте файл `.env` в корне проекта и добавьте туда токен вашего бота (получить можно у @BotFather):
`BOT_TOKEN=ваш_токен_здесь`

5. Запустите бота:
`python main.py`

## 🗄 Структура базы данных
При первом запуске скрипт автоматически создает локальный файл `coinkeeper.db` со следующей структурой таблицы `expenses`:
* `id` (INTEGER PRIMARY KEY)
* `user_id` (INTEGER) — Telegram ID пользователя
* `amount` (REAL) — Сумма транзакции
* `category` (TEXT) — Категория расхода
